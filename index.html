Simple Webpage with Favicon

This is a simple webpage that demonstrates how to implement a favicon. Look at the browser tab to see the star emoji favicon.

About Favicons

A favicon (short for "favorite icon") is a small icon associated with a website or webpage, displayed in various places such as:

  • Browser tabs
  • Bookmarks list
  • Browser history
  • Mobile home screen when saved

How This Favicon Works

This page uses two methods for the favicon:

  1. An SVG emoji method (modern browsers)
  2. A base64 encoded PNG fallback (older browsers)

Both methods are inline and don't require external files.
